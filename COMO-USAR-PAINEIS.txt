================================================================================
  TROUBLESHOOTING
================================================================================

BOTAO NAO EXECUTA:
  Causa: Navegador bloqueia execucao de arquivos locais
  
  Solucao 1: Use o menu terminal
    MENU-PRINCIPAL.bat
  
  Solucao 2: Execute os .bat diretamente
    Clique duas vezes no arquivo .bat no Explorer
  
  Solucao 3: Configure o navegador
    Chrome: Permitir arquivos locais
    Edge: Permitir downloads

PAINEL NAO ABRE:
  Causa: Arquivo HTML nao associado com navegador
  
  Solucao: Clique com botao direito ‚Üí Abrir com ‚Üí Chrome/Edge

TERMINAL FECHA RAPIDO:
  Causa: Script terminou
  
  Solucao: Os scripts tem pause automatico
    Se fechar rapido, ha erro - veja o log

ERRO "Python nao encontrado":
  Causa: Python nao instalado ou nao no PATH
  
  Solucao: Instale Python
    https://www.python.org/downloads/
    Marque "Add to PATH" na instalacao

================================================================================
  COMPARACAO: ANTES VS AGORA
================================================================================

ANTES (Linha de Comando):
  1. Abrir terminal
  2. Navegar ate pasta
  3. Digitar comando exato
  4. Lembrar nomes dos arquivos
  5. Riscos de erro de digitacao

AGORA (Botoes Clicaveis):
  1. Abrir painel HTML
  2. Clicar no botao
  3. Pronto!

ECONOMIA DE TEMPO:
  Linha de comando: ~1-2 minutos por tarefa
  Botoes clicaveis: ~5 segundos por tarefa
  
  RESULTADO: 12-24x mais rapido!
